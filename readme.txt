高阶函数：JavaScript的函数其实都指向某个变量。既然变量可以指向函数，函数的参数能接收变来那个
那么一个函数就可以接收另一个函数作为参数，这种函数就称为高阶函数。

闭包:在一个函数outer里面又定义了一个函数inner,inner函数作为outer函数的返回值暴露出去，
inner函数可以使用outer函数的私有变量，但是在outer函数外面，这些私有变量是不能被直接操作的，
这种结构叫闭包。
